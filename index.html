Sponge-js

model-driven geo enriched server for multi dimensional data

Download this project as a .zip file Download this project as a tar.gz file

Sponge-JS is model-driven geo enriched server for multi dimensional data.

It's intended to use for scientific calculations.

Sponge-JS delivers a webservice for

  • Data import, storage, correction, aggregation
  • Model lib: Environment for computing models with a lot of helpful functions
  • Managing background computing jobs
  • Working with geo data, connect data and maps, clip and create maps on the fly

It is used in this projects: AgroHyd - Welternährung und Wasserressourcen: eine agrarhydrologische Perspektive Leibniz Graduate School Landwirtschaftliche Verfahren: Potentiale und Kosten für die Treibhausgasminderung (LandPaKT)
